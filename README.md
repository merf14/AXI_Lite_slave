# AXI-Lite Slave с поддержкой конвейеризации

Реализация интерфейса AXI-Lite с конвейеризацией - для каждого канала используется FIFO для хранения данных о нескольких транзакицях. 

**rtl/axi_module.sv** - основной модуль AXI-Lite <br>
**rtl/fifo.sv** - универсальный модуль FIFO <br>
**tb/testbench.sv** - тестбенч с генерацией и проверкой транзакций чтения и записи

Для запуска симуляции в QuestaSim

```make```

Для очистки временных файлов

```make clean```
